# Agents Entry Point

Цей файл — універсальна точка входу для будь-якого AI агента (Claude Code, Cursor, Codex, OpenCode, Windsurf). Усі вони конвенцією читають `AGENTS.md` у корені проекту.

Основна інструкція — у @CLAUDE.md. Цей файл підтримує кросплатформенність.

## Швидкий старт для агента

1. @CLAUDE.md — повні інструкції проекту
2. @docs/state/hot.md — поточний стан і наступні кроки
3. @docs/project/structure.md — де що лежить

## Ритуали

- Нова сесія → `/start` (або прочитай hot.md вручну)
- Кінець сесії → `/handoff` (онови hot.md)
- Нова фіча → brainstorming → proposal → tasks → implementation
